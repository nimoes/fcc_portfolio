About Myself

After graduating from college, I pursued my newly developed interest in information security. Since then, I worked as a security operations center analyst, performing blue team operations and incident response. I enjoyed spending extra time studying logs, traffic patterns, and user behavior so that I could build and improve monitoring capabilities for my colleagues. Last year, I narrowed down my interest, and started dedicating more of my time studying web application and web application security assessments. This is when I began to realize that learning how to program is essential, not only in fostering my cyber security endeavor, but also in writing better, secure code that I hope to achieve in the near future. Please check out some of my works I have done so far, whether they were accomplished via FreeCodeCamp, or my personal research, and feel free to provide any constructive feedback!
